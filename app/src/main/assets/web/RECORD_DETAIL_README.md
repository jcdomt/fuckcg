# 运动记录详情功能说明

## 功能概述
在"运动记录"tab 中，用户可以点击任何一条运动记录来查看其详细信息。

## 文件组织

### 核心文件（改动）
- **work.html** - 工作页面（修改：引入 record-detail.js）
- **app.js** - 主应用逻辑（修改：renderRecords 函数添加点击事件）
- **style.css** - 全局样式（增加：详情页面和可点击记录的样式）

### 新增文件
- **record-detail.html** - 运动记录详情页面
  - 独立的详情页面，显示完整的运动信息
  - 包含 4 个信息卡片：时间信息、运动数据、审核状态、路线信息
  - 支持返回按钮和浏览器后退导航

- **record-detail.js** - 运动记录详情模块
  - 提供 RecordDetailModule 全局对象
  - 方法：`openDetail(record)` - 打开详情页面
  - 方法：`closeDetail()` - 关闭详情页面
  - 方法：`formatRecord(record)` - 格式化记录数据
  - 方法：`getStatusText(record)` - 获取状态文字
  - 方法：`getStatusClass(record)` - 获取状态样式类
  - 方法：`getSummary(record)` - 生成摘要文本

## 用户交互流程

```
运动记录列表页面
       ↓ (点击某条记录)
       ↓
record-detail.html (详情页面)
       ↓ (点击返回/浏览器后退)
       ↓
运动记录列表页面
```

## 技术实现

### 数据传递
- 使用 `sessionStorage` 在页面间传递数据
- 记录数据序列化为 JSON 字符串保存在 `sessionStorage.currentRecord`
- 详情页面加载时从 `sessionStorage` 读取数据

### 样式特性
- **可点击反馈**：鼠标悬停时卡片有蓝光效果和上浮动画
- **响应式设计**：在移动端自适应调整布局
- **深色主题**：与全应用风格一致

### 展示内容

#### 时间信息卡片
- 开始时间
- 结束时间
- 活动时长

#### 运动数据卡片
- 距离 (km)
- 平均速度 (km/h)
- 平均配速
- 步数
- 卡路里 (kcal)

#### 审核状态卡片
- 有效性（有效/无效/待审核）
- 审核状态代码
- 提示信息

#### 路线信息卡片
- 运动路线名称
- 运动类型
- 记录ID

## 扩展方向
